# Mobile Screenshot App

QXQS Lab includes a mobile-friendly Streamlit app for generating vertical signal screenshots.

The app is designed for Xiaohongshu posts, Douyin/Bilibili short-video screenshots, quick ticker case studies, and GitHub star call-to-action content.

## Install Web Dependencies

```bash
pip install -e ".[web]"
```

## Launch

```bash
qxqs web
```

Then open:

```text
http://localhost:8501
```

## Workflow

1. Enter a symbol such as `QQQ`, `NVDA`, `SPY`, or `TSLA`.
2. Choose a range.
3. Keep chart format as `xhs`.
4. Click `Generate chart`.
5. Download the generated screenshot.
6. Post it with a short explanation and GitHub CTA.

## Suggested CTA

```text
This chart was generated by QXQS Lab, an open-source low-frequency signal toolkit.
GitHub: nick-liyao/qxqs-lab
Star it if you want more mobile chart templates and Futu/Moomoo import guides.
```

## Boundary

The mobile app is for research, education, and content creation. It is not financial advice and does not provide trading instructions.
