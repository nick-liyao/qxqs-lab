# Xiaohongshu Posts

## Post 1

我把红绿线低频信号做成了一个开源 Python 工具。

项目叫 QXQS Lab。它不是荐股，也不是自动交易，而是一个本地优先的低频量化研究工具：导入日线数据后，生成红绿趋势线、B/BBB/S/SS/HOT/SSS 分档信号、低位修复筛选和回测报告。

GitHub 搜 `qxqs-lab`，如果你想要 watchlist 批量扫描和富途/牛牛导入教程，可以点个 Star。

## Post 2

输入一个美股代码，自动生成小红书 4:5 信号图。

我给 QXQS Lab 加了一个手机端截图页。输入 `QQQ`、`NVDA`、`TSLA` 这类标的，就能生成红绿趋势线、B/SS/HOT 等信号标签，还有简单回测摘要。

它不是荐股，也不是自动交易，只是把低频复盘流程做成可以复现的开源工具。

GitHub 搜 `qxqs-lab`，Star 到下一个阶段我会继续加 watchlist 批量扫描和富途/牛牛导入教程。
